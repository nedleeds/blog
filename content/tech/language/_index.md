## Programming Languages

프로그래밍 언어에 관한 다양한 주제를 다룹니다. 각 언어의 특성, 사용법, 그리고 관련된 기술을 설명합니다.

### 주요 언어

- [**C++**]({{<relref "cpp/_index.md">}}): 성능 중심의 언어, 멀티스레딩, 메모리 관리
- **Python**: 간결한 문법과 강력한 라이브러리 지원
- **JavaScript**: 웹 개발에서 핵심적인 언어
- **HTML & CSS**: 웹 페이지 구조 및 스타일링
